## Offline Support

### Using the App Offline
AA FairShare works even when you're offline or have a poor internet connection:

1. **Viewing Expenses**: Previously loaded expenses are available offline
2. **Creating Expenses**: New expenses can be created offline and will sync when you're back online
3. **Editing**: Changes made offline will be synchronized automatically
4. **Reports**: Downloaded reports remain accessible offline

### Installing as a PWA
For the best offline experience, install AA FairShare as a Progressive Web App:

1. Chrome/Edge:
   - Click the install icon (➕) in the address bar
   - Select "Install AA FairShare"

2. iOS Safari:
   - Tap the share button
   - Select "Add to Home Screen"

3. Android Chrome:
   - Tap the menu (⋮)
   - Select "Add to Home Screen"

## Troubleshooting

### Common Issues

1. **App Feels Slow**:
   - Clear cache in Settings > Storage
   - Check your internet connection
   - Update to the latest version
   - Reinstall the PWA if installed

2. **Offline Mode Not Working**:
   - Ensure PWA is installed
   - Check storage permissions
   - Verify offline mode is enabled
   - Clear and update cache

3. **Sync Issues**:
   - Check internet connection
   - Verify account status
   - Try manual sync
   - Contact support if persistent
